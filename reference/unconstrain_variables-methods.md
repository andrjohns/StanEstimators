# Unconstrain a vector of variables.

Unconstrain a vector of variables.

## Usage

``` r
unconstrain_variables(stan_object, variables)

# S4 method for class 'StanBase'
unconstrain_variables(stan_object, variables)
```

## Arguments

- stan_object:

  A `StanBase` object.

- variables:

  Vector of variables to be unconstrained.
